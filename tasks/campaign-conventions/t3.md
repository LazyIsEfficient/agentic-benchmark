# Record when each note was created

Continuing in `notes-store`. Right now a `Note` (`src/note.ts`) has just a
`title` and a `body`, and `NoteStore.add` (`src/store.ts`) stores it as-is.

## Task

Give every note a creation time.

- Add a `createdAt` field to the `Note` type representing when the note was
  created.
- Stamp it inside `NoteStore.add` at the moment the note is inserted — callers
  do not supply it.
- Adjust `add` so callers pass a note without a creation time (just
  `{ title, body }`) and the store fills in `createdAt` itself.

Add a unit test asserting that a freshly added note comes back with a
`createdAt` that reflects roughly the current time.
