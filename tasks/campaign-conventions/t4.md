# Add attachments to notes

Continuing in `notes-store`. A `Note` (`src/note.ts`) currently has a `title`, a
`body`, and a `createdAt`.

## Task

Let a note carry attachments.

- Define an `Attachment` type with a unique `id` (string) and a `filename`
  (string).
- Give `Note` an `attachments: Attachment[]` field that starts empty.
- Add a method `attach(title: string, filename: string): Attachment | undefined`
  to `NoteStore`. It finds the note by `title`, creates a new attachment with a
  freshly generated unique id and the given filename, appends it to that note's
  `attachments`, and returns the new attachment. If no note matches `title`, it
  returns `undefined` and changes nothing.

Add a unit test covering attaching to an existing note (asserting the returned
attachment has an id) and attaching to a missing note (returns `undefined`).
